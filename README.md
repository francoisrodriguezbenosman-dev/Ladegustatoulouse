# La Degust' Toulouse

Static site built with Astro.

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Preview

```sh
npm run preview
```

## Project Structure

- `src/pages/` routes
- `src/components/` UI pieces
- `src/layouts/` shared layouts
- `src/styles/` global styles
- `public/` static assets
